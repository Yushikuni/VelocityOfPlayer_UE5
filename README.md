# VelocityOfPlayer_UE5

This component is in BP for UE5.4.4

### How to aply:
1. Make sure that this component is included in charcter
	1. Get on main character
	2. Click on Add button 
	3. Add component -> BP_Velocity_CharacterMovement 
2. Build 
3. Run 


### Notes:
The `BP_Velocity_CharacterMovement` component contains two functions.

1. **GetOwner_void**
	- Returns a `void`
	- Setting a owner for printing

2. **printSpeed**
	- Returns `void`
	- Prints the velocity on the screen and on `UE_LOG`
