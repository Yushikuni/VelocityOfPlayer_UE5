# VelocityOfPlayer_UE5

This component is in C++ for UE5.4.4

### How to aply:
1. Make sure that this component is included in charcter
	1. Get on main character
	2. Click on Add button 
	3. Add component -> Velocity_CharacterMovement 
2. Build 
3. Run 


### 
This Velocity_CharacterMovement component have 2 methods.
1. GetSpeed() const
	1. return float

2. printVelocity()
	1. return void		

	
Get speed give me a float for calculating velocity on character.
Print velocity printing velocity on screen and on UE_LOG could be deleted leter.
